# hosh
## すること
ある文字列をスレッドタイトルに含むスレを検索し保守する
## 設定できるもの
* 名前欄
* メール欄
* 本文欄
* 保守間隔時間（秒）
* 検索するスレッドタイトル
## 必要なもの
* Python（3.9で動作確認）
* ChromeDriver（86で動作確認）
